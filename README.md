# 🎉 Rifa Beneficente da Família

Site estático para gerenciamento de números de rifa beneficente.

## Estrutura

| Arquivo | Descrição |
|---|---|
| `index.html` | Página pública — visitantes escolhem números e fazem reservas |
| `public.js` | Lógica da página pública |
| `app.js` | Estado compartilhado via `localStorage` |
| `style.css` | Estilos |
